## What MIT means here

The MIT License is a permissive open-source license.

It allows other people to:

- use the code
- copy the code
- modify the code
- distribute the code
- use it privately
- use it commercially

## Main condition

If someone copies or redistributes substantial parts of the software, they need to keep the copyright notice and license text.

In this project that means they should keep something like:

```text
Copyright (c) 2026 Daniel Sievers
MIT License
```

and the MIT license text.

## What this means in practice

People may fork, study, adapt or reuse the project, including commercially, as long as they keep the license and copyright notice.

The license does not mean that the author guarantees correctness, financial accuracy, data safety or fitness for a specific purpose.

## Why this license was chosen

The project is meant as a public portfolio project. The MIT License makes it easy to inspect, learn from and reuse while keeping attribution and limiting warranty/liability expectations.

## Alternative

If the goal were to allow viewing but not broad reuse, a more restrictive license would be needed. MIT is intentionally open and permissive.
