# Vercel deployment guide

This project can be deployed as a static site.

## Settings

```text
Framework Preset: Other
Build Command: empty
Output Directory: empty / root
Install Command: empty or npm install
```

## After deployment

1. Open the live URL.
2. Load all demo profiles.
3. Test portfolio and purchase history.
4. Take screenshots.
5. Add the live URL and screenshots to README.

No backend is required for the public demo.
