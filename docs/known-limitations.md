# Known limitations

- No financial advice. Portfolio data is fictional in the public demo.
- Automatic price updates support selected crypto and metals.
- Stock/ETF quote updates require a locally stored Alpha Vantage API key and depend on symbol availability.
- European ETF symbols may need manual provider-symbol adjustment.
- Manual price setting remains the reliable fallback.
- FX lookup uses a public no-key endpoint with local fallback values.
- No backend, login, multi-user collaboration or cloud sync is included in the public demo.
- Test coverage is limited to syntax checks and a public-release smoke check.
